# Che facciamo oggi, amore?

Mini sito statico romantico/ironico per scegliere un pomeriggio insieme tramite emoji.

## Funzioni

- schermata iniziale con pulsante “No” che scappa;
- scelta del mood tramite emoji;
- card finale con piano, budget, dress code e piano B meteo;
- bottone WhatsApp con messaggio precompilato;
- confetti leggero senza librerie esterne;
- responsive per telefono.

## Pubblicazione su GitHub Pages

1. Crea una nuova repository su GitHub chiamata `che-facciamo-oggi-amore`.
2. Carica questi file nella root della repo:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. Vai su **Settings → Pages**.
4. In **Build and deployment**, scegli:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Salva. Dopo poco il sito sarà online.

## Personalizzazione rapida

Modifica l’array `plans` in `script.js` per cambiare le proposte, i testi e i budget.
